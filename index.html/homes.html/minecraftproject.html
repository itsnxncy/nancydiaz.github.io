Minecraft Project

Hello! You have chosen the BUNNY! This page is about my Minecraft Art. I chose to build Totoro because I love the movie My Neighbor Totoro. I wanted to print out something that holds meaning to me. The Studio Ghibli films are movies that feel like home to me. Plus, this character is really cute and I love the vibe it gives off. It can be used as decoration anywhere in my home because of the coloring. The coloring is neautral and earthy, which I love!
